# v1.0.0-rc5.7

## Goal

Include imported functions in static autocomplete analysis and document imported-function access to global runtime variables.

## Required

- [x] Include globally imported functions in `Analyze` results.
- [x] Include alias-imported functions as qualified callable names.
- [x] Preserve autocomplete symbols while an imported file is missing during editing.
- [x] Confirm imported functions can create or update global `RuntimeContext` variables.
- [x] Document the imported-file top-level statement restriction.
- [x] Allow hosts to disable the iteration limit for `while` and `foreach`.
- [x] Keep stop and cancellation checks active when the iteration limit is disabled.

## Validation

- [x] Add regression coverage for global and alias-imported autocomplete functions.
- [x] Add regression coverage for an imported function writing `global.A`.
- [x] Add sync and async regression coverage for unlimited `while` and `foreach` execution.
