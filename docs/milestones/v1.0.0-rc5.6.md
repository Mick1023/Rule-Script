# v1.0.0-rc5.6

## Goal

Ship a hotfix so RuleScript debug and exception messages identify the source file associated with a line and column.

## Required

- [x] Include source file paths in exception messages when `SourceFile` is available.

- [x] Ensure imported-file syntax errors report the imported file path in the message.
- [x] Ensure runtime error event messages include the same file path as the event location.

## Validation

- [x] Add regression tests for imported-file syntax diagnostics.
- [x] Add regression tests for `ExecuteFile` main-file syntax diagnostics.
