# RuleScript v1.10.1 API Reference

This reference covers the public API added in v1.10.1 for source-file-aware editor analysis. For HostTrigger runtime APIs, see the v1.10.0 API reference.

## `RuleScriptEngine`

Namespace: `RuleScript.Core.Runtime`

### `Analyze`

```csharp
public RuleScriptAnalysisResult Analyze(
    string script,
    string sourceFile);

public RuleScriptAnalysisResult Analyze(
    string script,
    string sourceFile,
    int line,
    int column);
```

Strictly analyzes script text using `sourceFile` to resolve relative imports. Relative imports are based on the source file directory; relative source file paths are resolved from `WorkingDirectory`.

### `TryAnalyze`

```csharp
public RuleScriptAnalysisAttempt TryAnalyze(
    string script,
    string sourceFile);

public RuleScriptAnalysisAttempt TryAnalyze(
    string script,
    string sourceFile,
    int line,
    int column);
```

Best-effort analyzes script text using `sourceFile` to resolve relative imports. This is intended for editor buffers where the current text may be incomplete but still belongs to a known project file.
