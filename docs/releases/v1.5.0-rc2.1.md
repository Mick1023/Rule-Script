# Rule Script v1.5.0-rc2.1

This prerelease fixes user-function recursion, which had been rejected before the configured call-depth limit could be applied.

- Recursive functions can now complete normally within `MaxCallDepth`.
- Infinite recursion is stopped by `MaxCallDepth` when `CallDepthLimitEnabled` is enabled.
- Disabling `CallDepthLimitEnabled` allows finite recursion beyond the configured depth.
- The fix applies to both `Execute` and `ExecuteAsync`.
