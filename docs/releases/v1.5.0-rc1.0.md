# Rule Script v1.5.0-rc1.0

This prerelease publishes the first release candidate of Rule Script v1.5.0 for NuGet integration testing.

- Adds script-level `parallel` and `task` blocks.
- Supports parallel statements and ordered parallel expression results.
- Integrates cooperative cancellation with `RuleScriptEngine.Stop()` and runtime limits.
- Isolates task-local scopes while synchronizing shared runtime globals.
- Requires host functions used by parallel tasks to be explicitly marked thread-safe.

See [v1.5.0 release notes](v1.5.0.md) for syntax, behavior, and limitations.
