# RuleScript v1.0.0-rc5.1

## Patch Scope

- Fixes editor workflow regressions around `TryAnalyze` followed by execution.
- Clarifies synchronous and asynchronous runtime event handler behavior.

## Fixed

- `TryAnalyze` does not prevent later `Execute` calls on the same engine.
- `TryAnalyze` does not prevent later `ExecuteAsync` calls on the same engine.
- `TryAnalyze` does not prevent later `RuntimeEventHandler` notifications.
- `TryAnalyze` does not prevent later `RuntimeEventHandlerAsync` notifications.
- Synchronous `Execute` now ignores `RuntimeEventHandlerAsync` and continues running scripts. Async runtime event handlers only run during `ExecuteAsync` and `ExecuteFileAsync`.
