# RuleScript v0.5.0

## Added

- Added JSON built-in functions
- Added `JsonParse(value)`
- Added `JsonStringify(value)`
- Added `JsonGet(value, path)`
- Added `JsonSet(value, path, newValue)`
- Added integration with property access, index access, and `foreach`

## Behavior

- JSON objects become `Dictionary<string, object?>`.
- JSON arrays become `List<object?>`.
- JSON numbers become `decimal`.
- `JsonSet` mutates dictionaries and lists in place and returns the original object.

## Not Added

- No JSON literal syntax
- No `switch`
- No user-defined functions
- No async host functions

## Validation

- Build passed
- Tests passed
- NuGet package generated
