# RuleScript v0.7.1

## Improved

- Improved `ExecuteFile` diagnostics.
- Improved missing import diagnostics.
- Improved duplicate alias diagnostics.
- Improved unknown alias diagnostics.
- Improved missing module function diagnostics.
- Improved circular import diagnostics with import chains.
- Improved path normalization for imports.
- Added import file restrictions for imported modules.

## Path Resolution

- Relative `ExecuteFile` paths resolve from `RuleScriptEngine.WorkingDirectory`.
- If `WorkingDirectory` is `null`, `Environment.CurrentDirectory` is used.
- Absolute `ExecuteFile` paths ignore `WorkingDirectory`.
- Import paths resolve relative to the importing file.
- Equivalent normalized import paths share one loaded module.

## Diagnostics

- Missing `ExecuteFile` paths include `ExecuteFile`, the original path, and the resolved full path.
- Missing imports include `import`, the original import path, the importing file, and the resolved full path.
- Duplicate aliases include the duplicate alias and file path.
- Missing module functions include the alias, function name, and `function not found`.
- Circular imports include the import chain.
