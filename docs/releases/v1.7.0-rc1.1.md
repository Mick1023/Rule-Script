# RuleScript v1.7.0-rc1.1

RuleScript v1.7.0-rc1.1 is a hotfix release candidate for the v1.7 Navigation API.

## Hotfix

- Adds convenience source-position properties to `RuleScriptDefinitionInfo`:
  - `File`
  - `Line`
  - `Column`
  - `EndLine`
  - `EndColumn`
- Adds the same convenience source-position properties to `RuleScriptReferenceInfo`.
- Keeps the existing `Range` and `SelectionRange` metadata intact for callers that need full source ranges.

## Compatibility and Scope

- No Rule Script syntax changes.
- No runtime behavior changes.
- No parser, binder, type checker, or module system rewrite.
- Navigation metadata remains a `RuleScript.Core` API surface only.
