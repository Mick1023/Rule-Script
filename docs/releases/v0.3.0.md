# RuleScript v0.3.0

## Added

- Array literals
- Array index access
- Object property access
- `Length(array)` support

## Syntax

```rulescript
var values = [1, 2, 3];
result = values[0];

var robot = GetRobot();
result = robot.Position.X;
```

## Runtime

- Arrays are represented internally as `List<object?>`.
- Indexes must be int-compatible values.
- Property access supports `Dictionary<string, object?>`, `IReadOnlyDictionary<string, object?>`, and public object properties.
- Dictionary keys are checked before public properties.

## Not Added

- No `foreach`
- No `switch`
- No user-defined functions
- No JSON syntax
- No async host functions

- Build passed
