# RuleScript v0.1.1

Release preparation for publishing `RuleScript.Core` to NuGet.org.

## Highlights

- GitHub Actions packaging workflow
- NuGet-ready package metadata
- XML documentation output
- README included in the NuGet package
- Symbol package output with `.snupkg`
- NuGet publish workflow using GitHub Secrets

## NuGet Package

Package ID:

```text
RuleScript.Core
```

Install after publishing:

```powershell
dotnet add package RuleScript.Core
```

## Publishing

The package workflow publishes only when:

- A tag matching `v*` is pushed
- Or the workflow is manually run with `publish=true`

Publishing uses the GitHub repository secret `NUGET_API_KEY`. No API key is stored in the repository.

## Known Limitations

- No loops
- No arrays
- No JSON/object literals
- No user-defined script functions
- No async host functions
