# RuleScript v0.6.0

## Added

- Added user-defined functions with `function` / `endfunction`
- Added `return` statement
- Added local function scope
- Added recursion protection
- Added user function lookup before host and built-in functions

## Behavior

- Function declarations are top-level only.
- Parameters and `var` declarations inside functions are local.
- Functions can read values from the global `RuntimeContext`.
- Assignment inside a function updates local variables first, then existing globals, otherwise creates a local variable.
- A function without an explicit return returns `null`.
- `return;` returns `null`.

## Function Lookup Order

1. User-defined functions
2. Host functions
3. Built-in functions

## Not Added

- No import or module system
- No classes
- No lambda syntax
- No async functions
- No `switch`
- No `try` / `catch`

- Build passed
- Tests passed
