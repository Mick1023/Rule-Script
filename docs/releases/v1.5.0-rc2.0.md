# Rule Script v1.5.0-rc2.0

This prerelease improves parallel debugging with session-wide breakpoint pauses.

- A breakpoint or step pause now establishes a shared debug barrier.
- All parallel tasks pause cooperatively at their next statement/runtime-event boundary.
- Other tasks no longer continue emitting execute-line or print events while the session is paused.
- Continue and Step Over release all paused tasks through the same directive.
- Stop releases and cancels every task waiting at the shared pause barrier.

Long-running Host Functions are not forcibly suspended. They join the pause after returning to the next cooperative statement boundary.
