# RuleScript v1.4.0-rc1.2

## Hotfix

- Fixed `RuleScriptAnalysisResult.UserFunctions` so imported modules expose only functions explicitly declared with `export`.
- Fixed `RuleScriptAnalysisResult.VisibleVariables` so imported `export const` declarations are visible with readonly and export metadata.
- Removed legacy implicit module exports from runtime execution. Imported functions and constants are now private unless explicitly declared with `export`.
- Updated module examples and regression fixtures to use explicit exports.

## Breaking Update

- Modules without explicit exports no longer expose their functions or constants through global or alias imports.
- Existing imported declarations must be migrated to `export function` or `export const`.
