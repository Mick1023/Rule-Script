# RuleScript v1.8.0-preview1.0

RuleScript v1.8.0-preview1.0 is a NuGet preview for testing the v1.8 maintenance refactor before a stable release.

This preview does not add Rule Script syntax, executable language constructs, or runtime behavior changes.

## Preview Scope

- Unified function metadata through `RuleScriptFunctionSymbol`.
- Function origin represented by `RuleScriptFunctionKind`.
- Obsolete compatibility adapters retained for `RuleScriptHostFunctionSymbol` and `RuleScriptBuiltinFunctionSymbol`.
- Shared function resolution for analysis, diagnostics, Go To Definition, and Find References.
- Centralized source range mapping through `RuleScriptSourceMapper`.
- Reusable editor document analysis through `RuleScriptLanguageService.AnalyzeDocument`.
- Per-document analysis reuse through `RuleScriptAnalysisCache`.

## Compatibility

- Existing runtime execution behavior is unchanged.
- Existing formatter output is unchanged.
- Existing diagnostics output is unchanged.
- Existing public APIs remain available.
- Legacy function symbol wrappers remain available but are obsolete; new integrations should prefer `RuleScriptFunctionSymbol`.

## Validation

The preview package is intended to be published by the existing GitHub Actions package workflow from tag `v1.8.0-preview1.0`.
