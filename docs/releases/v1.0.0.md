# RuleScript v1.0.0

## Added

- Added host runtime notifications through `RuleScriptEngine.RuntimeEventHandler`.
- Added source locations for host-observable runtime events.
- Added current-line notifications.
- Added `Print` host notifications.
- Added breakpoint registration and hit notifications.
- Added step-over execution notifications.
- Added host error notifications for syntax and runtime failures.
- Added `RuntimeContext.CurrentLocation`.

## Diagnostics

- `RuleScriptException` now exposes `SourceFile` when available.
- Syntax and runtime error notifications include file, line, column, token text, and message when available.

## Validation

- Full test suite passed.
- Added runtime notification coverage for `Print`, current-line tracking, breakpoints, step execution, syntax errors, and imported-file runtime errors.
