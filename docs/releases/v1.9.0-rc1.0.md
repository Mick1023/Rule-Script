# RuleScript v1.9.0-rc1.0

RuleScript v1.9.0-rc1.0 is the first release candidate for strongly typed user functions, overload analysis, and runtime overload dispatch.

## Highlights

- Explicit function return types with `-> type` syntax.
- Stronger return type diagnostics for mismatched or missing returns.
- User function overload declarations by parameter signature.
- Shared overload resolution across user, host, and built-in functions.
- Runtime overload dispatch with exact matches preferred before `any`.

## Compatibility And Scope

- Existing function syntax remains supported.
- Existing scripts continue to run.
- Runtime does not perform automatic type conversion for overload selection.
- No generics, nullable type syntax, delegates, lambdas, or interfaces were added.

## Validation

This release candidate is covered by tests for:

- Parser support for explicit return types.
- Old syntax compatibility.
- Declared return type metadata.
- Return type mismatch diagnostics.
- Missing return warnings.
- Duplicate function signatures.
- Unified overload return type rules.
- No matching overload diagnostics.
- Ambiguous overload diagnostics.
- Runtime overload dispatch.
- User, host, and built-in overload coexistence.
- v1.8 regression coverage.
