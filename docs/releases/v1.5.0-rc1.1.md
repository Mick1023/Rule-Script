# Rule Script v1.5.0-rc1.1

This prerelease fixes parallel Host Function thread-safety analysis through User Function call chains.

- Detects non-thread-safe Host Functions reached indirectly from parallel tasks.
- Supports multi-level User Function call chains.
- Uses cycle-safe reachability analysis.
- Avoids false positives for sequential-only User Functions and thread-safe Host Functions.

See [v1.5.0 release notes](v1.5.0.md) for the complete feature overview.
