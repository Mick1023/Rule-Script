# Rule Script v1.6.0-rc1.4

This prerelease expands editor analysis metadata for v1.6.0 testing.

## Editor Metadata

Static analysis now exposes documentation comments for variables and constants through `RuleScriptVariableSymbol.Documentation`.

```rulescript
/// info1
var a = 1;

/// info2
const b = 0;
```

Editors can use the same `///` documentation style already supported by functions to show hover text and completion details for analyzed variables.

## Built-In Function Metadata

Built-in function analysis symbols include documentation text, so completion lists can show descriptions for runtime-provided functions.

## Compatibility

- Existing variable symbol constructor calls remain source compatible because the documentation field is optional.
- Existing executable syntax and runtime behavior are unchanged.
- Documentation comments remain analysis metadata and do not affect script execution.

## Validation

- Test suite: 572 passed, 0 failed, 0 skipped.
