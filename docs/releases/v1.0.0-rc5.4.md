# RuleScript v1.0.0-rc5.4

## Patch Scope

- Adds an explicit stop flow for `RuleScriptDebugSession`.

## Added

- Added `RuleScriptDebugSession.Stop()`.
- Added an internal stop cancellation token for each debug run.
- Debug runs now link host-provided cancellation with the session stop token.

## Behavior

- `Stop()` cancels the current debug run.
- `Stop()` releases breakpoint and step pauses.
- Stopped debug runs complete as `OperationCanceledException`.
- `WaitForPauseAsync` completes as canceled after `Stop()`.
- `Stop()` is idempotent and can be called before, during, or after a run.
- Runtime event handlers are restored after normal completion, errors, cancellation, or stop.
