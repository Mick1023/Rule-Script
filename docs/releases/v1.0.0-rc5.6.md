# RuleScript v1.0.0-rc5.6

## Patch Scope

- Improves debug and exception messages by including the source file when available.

## Fixed

- Syntax and runtime exception messages now include `SourceFile` after the engine assigns it.
- Syntax errors in imported files now identify the imported `.rules` file in the thrown message.
- Runtime error event messages now include the same file information exposed through `RuleScriptRuntimeEvent.Location.File`.

## Validation

- Added coverage for syntax errors in imported files.
- Added coverage for syntax errors in main files executed with `ExecuteFile`.
- Added coverage for runtime error event messages including source file paths.
