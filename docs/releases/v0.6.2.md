# RuleScript v0.6.2

## Added

- Added explicit global access syntax with `global.name`.
- Added global read support from script and user-defined functions.
- Added global assignment support with `global.name = value;`.

## Scope Rules

Normal variables:

- Read: local scope, then global `RuntimeContext`
- Write: local scope only inside functions

Explicit globals:

- Read: global `RuntimeContext` only
- Write: global `RuntimeContext` only

## Examples

```rulescript
var count = 0;

function Test():
    global.count = 100;
endfunction

Test();
result = global.count;
```

## Diagnostics

- Reading a missing `global.name` throws `RuntimeException`.
- Error messages include `global` and the variable name.
