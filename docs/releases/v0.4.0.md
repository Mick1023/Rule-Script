# RuleScript v0.4.0

## Added

- Added `foreach` / `endforeach`
- Added iteration over arrays, `object?[]`, `IEnumerable<object?>`, and strings
- Added `break` / `continue` behavior inside `foreach`
- Added max loop iteration safety for `foreach`

## Syntax

```rulescript
foreach item in items:
    statement*
endforeach
```

## Runtime

- Each item is written to `RuntimeContext` for the duration of the iteration.
- Existing iterator variables are restored after the loop.
- New iterator variables are removed after the loop.
- String iteration yields one-character strings.
- `break` exits the nearest loop.
- `continue` skips to the next iteration of the nearest loop.

## Not Added

- No `for` / `endfor`
- No `switch`
- No JSON syntax
- No user-defined functions
- No async host functions
