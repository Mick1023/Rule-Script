# Rule Script v1.6.0-rc1.0

This prerelease publishes the first release candidate of Rule Script v1.6.0 for NuGet integration testing.

- Adds `RuleScriptFormatter.Format` with canonical indentation, spacing, and block layout.
- Preserves line, multi-line, documentation, and region comments during formatting.
- Adds `/* ... */` multi-line comments with unterminated-comment diagnostics.
- Adds nested `#region` and `#endregion` metadata for editor folding.
- Associates `///` documentation comments with user-defined function symbols.
- Exposes region and function documentation through `RuleScriptLanguageService`.
- Preserves all language syntax and execution behavior through v1.5.0, including `parallel` and `task`.

See [v1.6.0 release notes](v1.6.0.md) for syntax, APIs, compatibility, and scope.
